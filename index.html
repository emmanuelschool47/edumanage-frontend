// What we offer

Everything your school needs.

Admin Control

Full control of all students, teachers, fees and reports. Password protected with unique admin code.

Teacher Panel

Teachers see and manage only their assigned class. Update marks, attendance and post homework.

Student Portal

Students login with their unique code and see only their own marks, attendance, fees and timetable.

Class Group

WhatsApp style class group. Teacher posts homework, images, videos. Students view only.

UPI Fee Payment

Students pay fees via PhonePe, GPay or Paytm directly. Upload screenshot for admin verification.

Push Notifications

Instant notifications for attendance, fees, holidays and updates. Works even when app is closed.

// Access levels

Three panels, one system.

Admin

Principal or school manager. Full access to everything. Login required every time for security.

  • Add / Remove Students and Teachers
  • View all students class-wise
  • Manage fee payments
  • Promote students to next class
  • Manage passout students
Teacher

Class teacher. Sees and manages only their assigned class. Stays logged in permanently on phone.

  • View their class students only
  • Mark daily attendance
  • Update subject marks
  • Post in class group
  • Send class notifications
Student

Logs in with unique code given by admin. Stays logged in permanently. Sees only personal data.

  • View own marks and grades
  • Check attendance percentage
  • Pay fees via UPI
  • View class timetable
  • View class group posts
